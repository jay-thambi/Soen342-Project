About the System

This system allows administrators to create and manage offerings, instructors to select offerings, and the public to view available lessons. Use the navigation links above to explore the available features.

Features

For Administrators

Administrators can create, update, and delete offerings. Click on "Create New Offering" to add new lessons or "View Offerings" to manage existing ones.

Create Offering Manage Offerings

For Instructors

Instructors can view and select available offerings. Use the "Instructor Portal" link to access your options and take on new classes.

Instructor Portal

For Public

The public can browse available offerings but must register to make a booking. Click on "View Offerings" to explore available classes.
